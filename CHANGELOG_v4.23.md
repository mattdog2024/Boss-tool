# BossTool v4.23

## 修复

- 串行执行老板模式切换，避免快速重复触发导致模式、窗口与网络状态不一致。
- 退出程序前等待并验证工作 IP 恢复；恢复失败时取消退出。
- 严格验证保险箱弹出结果，失败时保留挂载状态，避免误报成功。
- 不再记录含 BitLocker 密码的 PowerShell 命令，并安全处理密码中的单引号。
- 停止在注册表中保存可逆混淆的保险箱密码；密码改为仅在当前运行期间保留。
- 只管理本程序启动的记事本，不再强制关闭用户自己的记事本或未保存文档。
- 停止清空浏览器、最近文件、资源管理器和远程桌面的全局历史。
- 停止删除本程序未创建的 Windows 或域安全策略。
- 正确处理 PowerShell、netsh 和命令行子进程超时，不再将仍运行的任务视为成功。

## 兼容性说明

- 保险箱密码升级后需要在每次启动时重新输入。
- 老板键默认快捷键仍为 `Ctrl+Win+Alt`。
- 网络地址、MAC 随机化、窗口隐藏和保险箱挂载的既有配置保持不变。
